## coolq-rollgames-bot
基于酷Q的自助Roll游戏插件  
本插件基于慕晓飞大神开发的[`酷Q Python SDK`](https://gitee.com/muxiaofei/coolq_sdk_x/wikis/pages "酷Q SDK-X") 编写

## 介绍<br>
* 群友自助进行Roll游戏活动
* 支持同时进行多个Roll游戏活动
* 发送“查看命令”可查看所有命令
